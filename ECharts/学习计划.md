## 第一部分：核心概念 & API（09:00-10:30）

### 1.1 初始化三步曲

```js
// 1. 准备 DOM 容器（必须有宽高）
<div id="chart" style="width:600px;height:400px;"></div>

// 2. 初始化实例
const chart = echarts.init(document.getElementById('chart'));

// 3. 设置配置项
chart.setOption({ /* option */ });
```

**面试常问：init 做了什么？**
- 创建 ECharts 实例对象
- 绑定渲染引擎（Canvas 默认 / SVG 可选）
- 注册事件系统
- 返回可操作的实例

### 1.2 Option 三大核心结构

```
option
├── 组件 (components)   → title, tooltip, legend, toolbox, grid...
├── 坐标轴 (axis)       → xAxis, yAxis, polar, radar...
└── 系列 (series)       → 数据 + 图表类型
```

**面试口诀：「组件在外，轴在中间，系列在里」**

### 1.3 setOption 的三个参数

```js
chart.setOption(option, notMerge?, lazyUpdate?)
// notMerge: false(默认合并) / true(覆盖)
// lazyUpdate: false(默认立即更新) / true(不立即更新)
```

| 参数 | 默认 | 作用 |
|------|------|------|
| notMerge=false | 合并模式 | 新旧 option 深度合并 |
| notMerge=true | 替换模式 | 完全替换 |
| lazyUpdate=true | 延迟更新 | 批处理时用 |

### 1.4 响应式布局

```js
// 方式一：resize 手动
window.addEventListener('resize', () => chart.resize());

// 方式二：media query 自动
option = {
  media: [
    { query: { maxWidth: 500 }, option: { legend: { orient: 'vertical' } } }
  ]
};
```

### 1.5 销毁

```js
chart.dispose();  // 销毁实例，释放资源
// 面试重点：组件卸载时必须 dispose，否则内存泄漏
```

## 第二部分：常用图表类型（10:30-12:00）

### 2.1 必须掌握（6 种）

| 类型 | series.type | 核心配置 | 面试频率 |
|------|-------------|----------|----------|
| 柱状图 | `bar` | barWidth, barGap, stack | 极高 |
| 折线图 | `line` | smooth, areaStyle, step | 极高 |
| 饼图 | `pie` | radius, roseType, label | 高 |
| 散点图 | `scatter` | symbolSize, encode | 中 |
| 雷达图 | `radar` | indicator, shape | 中 |
| 仪表盘 | `gauge` | min, max, progress | 低 |

### 2.2 双 Y 轴（面试必考）

```js
option = {
  yAxis: [
    { type: 'value', name: '销售额' },
    { type: 'value', name: '增长率' }
  ],
  series: [
    { type: 'bar', data: [820, 932, 901] },
    { type: 'line', yAxisIndex: 1, data: [12, 18, 24] }
  ]
};
```

### 2.3 堆叠图

```js
series: [
  { type: 'bar', stack: 'total', data: [...] },
  { type: 'bar', stack: 'total', data: [...] }
]
// stack 值相同 = 同组堆叠
```

### 2.4 饼图进阶配置

```js
series: [{
  type: 'pie',
  radius: ['40%', '70%'],     // 环形图：内半径, 外半径
  roseType: 'area',            // 南丁格尔玫瑰图
  label: { show: true, formatter: '{b}: {d}%' },
  emphasis: { scaleSize: 10 }  // 高亮放大
}]
```

## 第三部分：交互 & 事件系统（13:00-14:30）

### 3.1 事件绑定

```js
// 鼠标事件
chart.on('click', (params) => {
  console.log(params.name, params.value, params.seriesName);
});

// 组件交互事件
chart.on('legendselectchanged', (params) => { /* 图例切换 */ });
chart.on('datazoom', (params) => { /* 缩放平移 */ });

// 解绑
chart.off('click');
```

### 3.2 Tooltip 自定义

```js
tooltip: {
  trigger: 'axis',    // item(单点) | axis(坐标轴) | none
  formatter: (params) => {
    // params 是数组（axis 模式）或对象（item 模式）
    return `${params[0].name}<br/>销量: ${params[0].value}`;
  }
}
```

### 3.3 Toolbox 工具箱

```js
toolbox: {
  feature: {
    saveAsImage: { title: '保存' },  // 下载图片
    dataView: {},                     // 数据视图
    dataZoom: {},                     // 区域缩放
    magicType: { type: ['line', 'bar', 'stack'] },  // 动态切换
    restore: {}                       // 还原
  }
}
```

### 3.4 DataZoom（区域缩放）

```js
dataZoom: [
  { type: 'slider', start: 0, end: 50 },    // 滑块型
  { type: 'inside', start: 0, end: 100 }     // 内置型（滚轮/拖拽）
]
```

### 3.5 VisualMap（视觉映射）

```js
visualMap: {
  type: 'continuous',  // continuous | piecewise
  min: 0, max: 100,
  inRange: { color: ['#50a3ba', '#eac736', '#d94e5d'] }
}
```

## 第四部分：数据处理 & dataset（14:30-16:00）

### 4.1 dataset 模式（推荐）

```js
// 传统写法（不推荐面试用这个回答）
series: [
  { type: 'bar', data: [1, 2, 3] },
  { type: 'line', data: [4, 5, 6] }
]

// dataset 写法（面试加分项）
dataset: {
  source: [
    ['product', 'sales', 'profit'],  // 第一行是维度名
    ['A', 820, 120],
    ['B', 932, 180],
    ['C', 901, 150],
  ]
},
series: [
  { type: 'bar', encode: { x: 'product', y: 'sales' } },
  { type: 'line', encode: { x: 'product', y: 'profit' } }
]
```

**为什么用 dataset？**
- 数据与图表分离
- 一份数据多图共用
- 支持数据变换（Transform）

### 4.2 Transform 数据变换

```js
dataset: [
  {
    id: 'raw',
    source: [/* 原始数据 */]
  },
  {
    transform: { type: 'filter', config: { value: 500 } }
  },
  {
    transform: { type: 'sort', config: { dimension: 1, order: 'desc' } }
  }
]
```

### 4.3 异步加载

```js
// 面试高频：如何加载异步数据？
// 答：先 setOption 空壳，再异步填充
chart.setOption({
  xAxis: { data: [] },
  series: [{ data: [] }]
});

const res = await fetch('/api/data');
chart.setOption({
  series: [{ data: res.data }]
});

// 或者 showLoading / hideLoading
chart.showLoading();
const res = await fetch('/api/data');
chart.hideLoading();
chart.setOption({ series: [{ data: res.data }] });
```

## 第五部分：性能优化 & 进阶（16:00-17:30）

### 5.1 大数据量优化

```js
// 渲染模式：大数据用 SVG
echarts.init(dom, null, { renderer: 'svg' });

// 采样折线图（数据降维）
series: [{
  type: 'line',
  sampling: 'lttb',  // 最大三角形桶采样
  large: true,        // 大数据量优化
  largeThreshold: 2000
}]

// 渐进式渲染
series: [{
  progressive: 400,         // 每帧渲染点数
  progressiveThreshold: 3000 // 超过多少启用渐进
}]
```

### 5.2 主题定制

```js
// 方式一：内置主题
echarts.init(dom, 'dark');

// 方式二：自定义注册
echarts.registerTheme('myTheme', {
  color: ['#5470c6', '#91cc75', '#fac858'],
  backgroundColor: '#fff',
  textStyle: {}
});
echarts.init(dom, 'myTheme');
```

### 5.3 多图表联动

```js
// group 实现联动
echarts.connect('group1');   // 实例关联
chart1.group = 'group1';
chart2.group = 'group1';
// 自动联动：tooltip、dataZoom 等
```

### 5.4 自定义系列（知道即可）

```js
series: [{
  type: 'custom',
  renderItem: (params, api) => {
    // 返回图形元素
    return { type: 'rect', shape: {...}, style: {...} };
  }
}]
```
