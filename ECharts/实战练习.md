## 🟢 需求一：销售数据看板（基础综合）

### 场景
做一个「电商月度销售看板」，包含一个组合图。

### 功能要求
1. **双 Y 轴组合图**：柱状图显示「销售额」（左轴），折线图显示「环比增长率%」（右轴）
2. 折线必须 **平滑（smooth）** 且带 **面积渐变**
3. tooltip **axis 触发**，同时显示两个系列的值，增长率要带 `%` 后缀
4. 点击柱子时，**弹出 alert** 显示该月份名称和销售额（事件系统）
5. 加一个 **toolbox**：支持保存图片 + 柱状/折线动态切换（magicType）

### 覆盖考点
- init / setOption / 双 Y 轴(yAxisIndex)
- series 混用 bar + line
- tooltip.trigger='axis' + formatter
- areaStyle 渐变（`echarts.graphic.LinearGradient`）
- 事件 `chart.on('click')`
- toolbox.feature

### 验收标准
- [ ] 左右两个 Y 轴刻度不同
- [ ] 折线平滑且有渐变填充
- [ ] hover 时 tooltip 同时显示两行，增长率带 %
- [ ] 点柱子能 alert 出正确数据
- [ ] toolbox 能下载图片、能切换图表类型

### 提示
```js
// 渐变填充
areaStyle: {
  color: new echarts.graphic.LinearGradient(0, 0, 0, 1, [
    { offset: 0, color: 'rgba(84,112,198,0.5)' },
    { offset: 1, color: 'rgba(84,112,198,0)' }
  ])
}
// 双轴绑定
series: [{ type:'bar', /*用左轴*/ }, { type:'line', yAxisIndex:1 /*用右轴*/ }]
```

## 🟡 需求二：dataset + 动态切换品类（数据驱动）

### 场景
后端返回多品类销售数据，用户可切换查看不同品类，要求 **数据与视图分离**。

### 功能要求
1. 必须用 **dataset.source** 管理数据（二维数组，第一行是维度名），**禁止**把数据写死在 series.data
2. series 用 **encode** 做维度映射
3. 页面上方放 3 个按钮：「全部 / 手机 / 家电」，点击后 **只更新 dataset 或用 transform.filter** 过滤数据，图表刷新
4. 切换时用 **合并模式 setOption**（体会 notMerge=false 不丢配置）
5. 加 **dataZoom**（slider 型），数据多时可缩放

### 覆盖考点
- dataset.source + encode（面试加分项）
- dataset transform: filter
- setOption 合并模式
- dataZoom 区域缩放
- 数据动态更新思路

### 验收标准
- [ ] series 里没有任何写死的 data 数组
- [ ] 切换按钮能正确过滤显示对应品类
- [ ] 切换后 title/坐标轴配置不丢失（验证合并模式）
- [ ] 底部有可拖动的 dataZoom 滑块

### 提示
```js
dataset: {
  source: [
    ['月份', '手机', '家电'],
    ['1月', 120, 80],
    // ...
  ]
}
// 方案A：切换时改 encode 的 y 维度
// 方案B：用 transform filter 过滤
dataset: [
  { source: [...] },
  { transform: { type: 'filter', config: { dimension: 'category', value: '手机' } } }
]
```

## 🔴 需求三：双图联动 + 大数据性能（进阶综合）

### 场景
一个监控页：上方折线图展示「10000 个数据点的实时曲线」，下方饼图展示「分段占比」，两图 **联动**。

### 功能要求
1. 折线图渲染 **10000+ 数据点**，必须开启 **性能优化**（sampling + large + 必要时 progressive）
2. 折线图配 **visualMap**：按数值高低映射颜色（低→绿，高→红，注意中国习惯：高/涨用红）
3. 上下两图用 **echarts.connect** 建立 group 联动（tooltip / dataZoom 同步）
4. 折线图带 **dataZoom inside**（滚轮缩放）
5. 页面提供「重新生成数据」按钮，重新随机生成并 **showLoading / hideLoading**
6. **离开页面/重新生成时正确 dispose**，避免内存泄漏

### 覆盖考点
- 大数据优化：sampling='lttb' / large / progressive
- visualMap 视觉映射（continuous）
- echarts.connect 多图联动
- dataZoom inside（滚轮）
- showLoading / hideLoading
- dispose 内存管理

### 验收标准
- [ ] 1万点折线滚动/缩放不卡顿
- [ ] 折线颜色随数值变化（红高绿低）
- [ ] 操作一个图，另一个图的 tooltip/缩放同步响应
- [ ] 滚轮能缩放折线
- [ ] 重新生成时有 loading 动画
- [ ] 反复重新生成不会内存暴涨（dispose 生效）

### 提示
```js
// 大数据
series: [{ type:'line', sampling:'lttb', large:true,
  progressive: 2000, progressiveThreshold: 5000 }]
// 联动
chartLine.group = 'g1'; chartPie.group = 'g1';
echarts.connect('g1');
// 生成 1 万点
const data = Array.from({length: 10000}, (_, i) => [i, Math.random()*100]);
```
