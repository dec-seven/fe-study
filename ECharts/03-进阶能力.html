Part 3: ECharts 进阶能力

1. Dataset 模式 + 数据过滤 Transform

面试加分项：dataset 做到数据与图表分离，一份数据多图共用。 transform: { type: 'filter' } 实现前端数据过滤。

2. 事件系统 — click / legendselectchanged / datazoom

点击图表元素查看事件输出 →
chart.on('click', fn) / chart.off('click') 解绑。 组件事件：legendselectchanged、datazoom 等。

3. Tooltip自定义 + Toolbox工具箱 + DataZoom缩放

Toolbox 内置保存图片/数据视图/动态切换/还原。 DataZoom 支持 slider(滑块) 和 inside(滚轮) 两种。

4. 主题 + visualMap 视觉映射

echarts.init(dom, 'dark') 或 echarts.registerTheme() 注册自定义主题。
